# CV_Danny

Página web tipo CV (Curriculum Vitae) hecha con HTML5 puro, como parte de la actividad de "Landing Page con HTML y GitHub Pages".

## Descripción

Este proyecto es una página estática que muestra mi información personal y académica como estudiante de Ingeniería en Sistemas en la Universidad Mariano Gálvez de Guatemala. Fue construida utilizando únicamente HTML5, sin CSS ni JavaScript, cumpliendo con los requisitos de la actividad.

## Contenido de la página

- Presentación personal
- Formación académica
- Intereses y habilidades
- Tabla de tecnologías
- Enlaces a la universidad y a LinkedIn
